# ACTG Sync

Selbst gehostetes Sync-Plugin fuer Obsidian (Vault-Sync, Freigaben, Geraeteverwaltung, Konflikt-Center, QR-Einrichtung).

**Installation ueber BRAT:** In Obsidian das Plugin "BRAT" installieren -> "Add beta plugin" -> `Nogareru/actg-synch` eintragen. Updates kommen ueber die Releases dieses Repos.

Dieses Repo enthaelt nur die Plugin-Releases; entwickelt wird intern.
